## 一、字符串的定义

用单引号、双引号或三引号包围起来的字符序列。

```python
s1 = 'hello, world!'
s2 = "你好，世界！❤️"
s3 = '''hello,
wonderful
world!'''
```

### 转义字符

| 转义符 | 含义 |
|---|---|
| `\n` | 换行 |
| `\t` | 制表符 |
| `\` | 反斜杠本身 |
| `\'` | 单引号 |
| `\"` | 双引号 |

```python
s1 = '\'hello, world!\''
s2 = '第一行\n第二行'
```

### 原始字符串

在字符串前加 `r` 或 `R`，不处理转义：

```python
s = r'C:\Users\name'   # 里面的 \n 不会变成换行
```

## 二、字符串的运算

### 拼接与重复

```python
s1 = 'hello' + ', ' + 'world'   # 'hello, world'
s2 = '!' * 3                    # '!!!'
```

### 比较运算

按字符的 Unicode 编码逐位比较。可用 `ord()` 查编码。

```python
print('A' < 'a')       # True  (65 < 97)
print('boy' < 'bad')   # False (第二个字符 o > a)
```

### 成员运算

```python
s = 'hello, world'
print('wo' in s)       # True
print('abc' not in s)  # True
```

### 获取长度

```python
s = 'hello, world'
print(len(s))   # 12
```

## 三、索引与切片

字符串是不可变类型，**不能通过索引修改字符**。

```python
s = 'abcdefg'

print(s[0])       # a        第一个
print(s[-1])      # g        最后一个
print(s[2:5])     # cde      索引2到4（包头不包尾）
print(s[:5])      # abcde    从头开始到索引4
print(s[1:])      # bcdefg   从索引1到末尾
print(s[::2])     # aceg     步长为2
print(s[::-1])    # gfedcba  反转字符串
print(s[-4:-1])   # def      倒数第4到倒数第2
```

> **口诀**：`[开始:结束:步长]`，**包头不包尾**。

## 四、字符串常用方法

### 1. 查询

| 方法 | 功能 | 找不到时 |
|---|---|---|
| `find(str)` | 从左查找子串第一次出现的位置 | 返回 `-1` |
| `rfind(str)` | 从右查找子串第一次出现的位置 | 返回 `-1` |
| `index(str)` | 同 find | **报错** `ValueError` |
| `rindex(str)` | 同 rfind | **报错** `ValueError` |

```python
s = 'hello, world!'
print(s.find('or'))      # 8
print(s.find('or', 9))   # -1
print(s.index('or'))     # 8
```

> **建议**：优先用 `find`，找不到不会报错。

### 2. 大小写转换

| 方法 | 功能 |
|---|---|
| `upper()` | 全部转大写 |
| `lower()` | 全部转小写 |
| `swapcase()` | 大小写互换 |
| `capitalize()` | 首字母大写，其余小写 |
| `title()` | 每个单词首字母大写 |

```python
s = 'hello, world!'
print(s.upper())       # HELLO, WORLD!
print(s.title())       # Hello, World!
```

> 字符串不可变，这些方法都**返回新字符串**，原字符串不变。

### 3. 对齐与填充

| 方法 | 功能 |
|---|---|
| `center(width, fillchar)` | 居中，两侧填充 |
| `ljust(width, fillchar)` | 左对齐，右侧填充 |
| `rjust(width, fillchar)` | 右对齐，左侧填充 |
| `zfill(width)` | 右侧对齐，左侧补 `0` |

```python
s = 'hello'
print(s.center(20, '*'))   # *******hello********
print(s.rjust(20))         #                hello
print('33'.zfill(5))       # 00033
print('-33'.zfill(5))      # -0033
```

### 4. 分割字符串

| 方法 | 功能 |
|---|---|
| `split(sep=None, maxsplit=-1)` | 按分隔符分割，返回**列表** |
| `rsplit(sep=None, maxsplit=-1)` | 从右边开始分割 |
| `splitlines()` | 按行分割 |
| `partition(sep)` | 找到第一个 sep，分成 3 部分，返回**元组** |
| `rpartition(sep)` | 从右边找第一个 sep 分割 |

```python
s = 'I#love#you#so#much'
print(s.split('#'))        # ['I', 'love', 'you', 'so', 'much']
print(s.split('#', 2))     # ['I', 'love', 'you#so#much']

s2 = 'I love you'
print(s2.split())          # ['I', 'love', 'you']
print('~'.join(s2.split())) # I~love~you
```

### 5. 合并与替换

| 方法 | 功能 |
|---|---|
| `join(iterable)` | 用该字符串作为分隔符，将序列合并成一个字符串 |
| `replace(old, new, count=-1)` | 把 old 替换成 new，count 限制次数 |

```python
words = ['I', 'love', 'you']
print('~'.join(words))     # I~love~you

s = 'hello, good world'
print(s.replace('o', '@'))     # hell@, g@@d w@rld
print(s.replace('o', '@', 1))  # hell@, good world
```

### 6. 判断字符串（返回 True/False）

| 方法 | 功能 |
|---|---|
| `isidentifier()` | 是否是合法标识符 |
| `isspace()` | 是否只有空白字符 |
| `isalpha()` | 是否全由字母组成 |
| `isdecimal()` | 是否全由十进制数字组成（**不含中文、罗马数字**） |
| `isdigit()` | 是否只包含数字（**不含中文数字**） |
| `isnumeric()` | 是否全由数字组成（**中文数字也算**） |
| `isalnum()` | 是否由字母和数字组成 |
| `islower()` | 是否全为小写 |
| `isupper()` | 是否全为大写 |
| `istitle()` | 是否标题化（每个单词首字母大写） |
| `isascii()` | 是否全为 ASCII 字符 |
| `isprintable()` | 是否全为可打印字符 |
| `startswith(str)` | 是否以 str 开头 |
| `endswith(str)` | 是否以 str 结尾 |

```python
s = 'abc123456'
print(s.isdigit())   # False
print(s.isalpha())   # False
print(s.isalnum())   # True
print(s.startswith('abc'))  # True
```

### 7. 修剪操作（去两端字符）

| 方法 | 功能 |
|---|---|
| `strip(chars=None)` | 去掉左右两边指定字符（默认空白） |
| `lstrip(chars=None)` | 去掉左边指定字符 |
| `rstrip(chars=None)` | 去掉右边指定字符 |

```python
s = '   jackfrued@126.com  '
print(s.strip())               # jackfrued@126.com
print('~你好，世界~'.lstrip('~'))  # 你好，世界~
```

## 五、格式化输出

### f-string（推荐，Python 3.6+）

```python
a = 321
b = 123
print(f'{a} * {b} = {a * b}')
```

### 常用格式化规则

| 占位符 | 例子 | 结果 | 说明 |
|---|---|---|---|
| `{:.2f}` | `f'{3.14159:.2f}'` | `3.14` | 保留2位小数 |
| `{:+.2f}` | `f'{3.14159:+.2f}'` | `+3.14` | 带符号保留2位 |
| `{:.0f}` | `f'{3.14159:.0f}'` | `3` | 不带小数 |
| `{:0>10d}` | `f'{123:0>10d}'` | `0000000123` | 左补0，宽度10 |
| `{:>10d}` | `f'{123:>10d}'` | `       123` | 左补空格，右对齐 |
| `{:<10d}` | `f'{123:<10d}'` | `123       ` | 右补空格，左对齐 |
| `{:,}` | `f'{123456789:,}'` | `123,456,789` | 千分位逗号 |
| `{:.2%}` | `f'{0.123:.2%}'` | `12.30%` | 百分比 |
| `{:.2e}` | `f'{123456789:.2e}'` | `1.23e+08` | 科学计数法 |

## 六、编码与解码

```python
a = '骆昊'
b = a.encode('utf-8')     # 编码为字节串
c = a.encode('gbk')

print(b)                  # b'\xe9\xaa\x86\xe6\x98\x8a'
print(b.decode('utf-8'))  # 骆昊
```

> 编码和解码方式必须一致，否则会乱码或报错。

## 七、遍历字符串

```python
s = 'hello'

# 方式一：按索引
for i in range(len(s)):
    print(s[i])

# 方式二：直接遍历元素（推荐）
for ch in s:
    print(ch)
```

## 总结

| 类别 | 重点 |
|---|---|
| 定义 | 单/双/三引号，`r` 原始字符串，不可变 |
| 索引切片 | `[开始:结束:步长]`，包头不包尾，不可赋值修改 |
| 查询 | `find`（返回-1）比 `index`（报错）安全 |
| 分割合并 | `split` 得列表，`join` 合并列表 |
| 判断 | `isdecimal` < `isdigit` < `isnumeric` 范围依次扩大 |
| 格式化 | f-string 最简洁，`{变量:格式}` 控制显示 |
