## 29. 字典（Dict）简介

字典是 Python 中以 **键值对（Key-Value）** 形式保存数据的容器，用花括号 `{}` 表示。通过**键（Key）**可以快速找到对应的**值（Value）**。

跟现实中的《新华字典》一样：每个字（键）都有对应的解释（值），合在一起就是一条条目。

```python
person = {
    'name': '王大锤',
    'age': 55,
    'height': 168,
    'weight': 60,
}
print(person['name'])   # 输出：王大锤
```

### 为什么需要字典？

用列表/元组存一个人的信息时，取数据得记住下标位置：

```python
# 列表 — 取手机号得先记住是第几个
p_list = ['王大锤', 55, 168, 60, '成都市武侯区...', '13122334455', '13800998877']
print(p_list[5])  # 得记住下标5才是手机号

# 字典 — 直接用键取值，语义清晰
p_dict = {'name': '王大锤', 'tel': '13122334455', 'emergency_tel': '13800998877'}
print(p_dict['tel'])  # 直接写键名就行
```

| 操作       | 写法                    | 说明               |
|----------|-----------------------|------------------|
| 取值       | `dict[键]`              | 通过键获取对应的值        |
| 修改       | `dict[键] = 新值`         | 键存在则修改值           |
| 新增       | `dict[新键] = 值`         | 键不存在则新增一组键值对     |
| 删除       | `del dict[键]`           | 按键删除整组键值对         |

> ⚠️ **注意：字典的键必须是不可变类型**（int、float、str、tuple）。列表、集合、字典不能作为键，但都可以作为值。

## 30. 字典的 5 种创建方式

### 方式 1：字面量 `{}`（最常用）

```python
xinhua = {
    '麓': '山脚下',
    '路': '道，往来通行的地方',
    '潞': '潞水，水名',
}
print(xinhua)
```

### 方式 2：`dict()` 构造器

```python
person = dict(name='王大锤', age=55, height=168)
print(person)  # {'name': '王大锤', 'age': 55, 'height': 168}
```

### 方式 3：`zip()` 压缩两个序列

把两个序列一一配对组成键值对：

```python
items = dict(zip('ABCDE', '12345'))
print(items)  # {'A': '1', 'B': '2', 'C': '3', 'D': '4', 'E': '5'}

items2 = dict(zip('ABCDE', range(1, 10)))
print(items2)  # {'A': 1, 'B': 2, 'C': 3, 'D': 4, 'E': 5}
```

### 方式 4：字典生成式（推荐，简洁高效）

```python
# 创建 {数字: 数字的立方} 的字典
cubes = {x: x ** 3 for x in range(1, 6)}
print(cubes)  # {1: 1, 2: 8, 3: 27, 4: 64, 5: 125}

# 带条件的生成式（只保留偶数）
even_cubes = {x: x ** 3 for x in range(1, 6) if x % 2 == 0}
print(even_cubes)  # {2: 8, 4: 64}
```

### 5 种创建方式对比

| 方式          | 写法                         | 适用场景            |
|-------------|----------------------------|-----------------|
| 字面量 `{}`   | `{'a': 1, 'b': 2}`          | 已知具体键值对，最常用      |
| `dict()`     | `dict(a=1, b=2)`            | 键是合法变量名时        |
| `zip()`      | `dict(zip(keys, values))`    | 有两个现成的序列要配对时    |
| 字典生成式       | `{k: v for k, v in ...}`      | 需要按规则批量生成时      |
| `fromkeys()` | `dict.fromkeys(keys, value)` | 所有键共用同一个默认值时   |

## 31. 成员运算与索引运算

### 成员运算 `in` — 判断键是否存在

```python
person = {'name': '王大锤', 'age': 55, 'height': 168}

print('name' in person)   # True  → 键存在
print('tel' in person)    # False → 键不存在
```

> ⚠️ 注意：`in` 只判断**键**，不判断值。

### 索引运算 `dict[键]` — 获取或设置值

```python
person = {'name': '王大锤', 'age': 55, 'height': 168}

# 取值
print(person['name'])     # 王大锤

# 修改已有的键
person['age'] = 25
print(person['age'])      # 25

# 新增不存在的键
person['tel'] = '13122334455'
person['signature'] = '你的男朋友是一个盖世垃圾'
print(person)
```

⚠️ **索引的坑：键不存在会报错 `KeyError`！**

```python
print(person['sex'])
# KeyError: 'sex'  ← 程序直接崩溃！

# 安全做法：先用 in 判断
if 'sex' in person:
    print(person['sex'])
else:
    print("该键不存在")
```

### 循环遍历

```python
person = {'name': '王大锤', 'age': 25, 'height': 178}

# for 默认遍历的是键（keys）
for key in person:
    print(f'{key}: {person[key]}')
# 输出：
# name: 王大锤
# age: 25
# height: 178
```

## 32. get() — 安全取值（重点！）

`get()` 方法在键不存在时**不会报错**，而是返回 `None` 或指定的默认值。这是它和索引运算最大的区别。

```python
person = {'name': '王大锤', 'age': 25, 'height': 178}

# 键存在 → 正常返回值
print(person.get('name'))        # 王大锤

# 键不存在 → 返回 None（不会崩溃）
print(person.get('sex'))         # None

# 键不存在 → 返回指定的默认值
print(person.get('sex', '未知')) # 未知
```

### get() vs 索引运算 对比（必考）

| 方式                | 键存在时    | 键不存在时         | 推荐场景     |
|-------------------|-----------|------------------|------------|
| `dict[键]`          | 返回对应值    | ❌ **KeyError 崩溃** | 确定键一定存在时 |
| `dict.get(键)`      | 返回对应值    | ✅ 返回 None / 默认值 | 不确定键是否存在时 |

> 记忆口诀：**确定有就用 `[]`，不确定就用 `get()`**

## 33. keys()、values()、items()

三个方法分别获取字典的键、值、键值对：

```python
person = {'name': '王大锤', 'age': 25, 'height': 178}

# keys() → 所有的键
print(person.keys())    
# dict_keys(['name', 'age', 'height'])

# values() → 所有的值
print(person.values())  
# dict_values(['王大锤', 25, 178])

# items() → 所有的键值对（每个是一个二元组）
print(person.items())   
# dict_items([('name', '王大锤'), ('age', 25), ('height', 178)])
```

### 用 items() 同时遍历键和值（最常用）

```python
for key, value in person.items():
    print(f'{key}\t→ {value}')
# 输出：
# name	→ 王大锤
# age	→ 25
# height	→ 178
```

| 方法        | 返回内容             | 典型用法                  |
|-----------|--------------------|-------------------------|
| `dict.keys()`   | 所有的键               | `for key in d.keys()`     |
| `dict.values()` | 所有的值               | `for val in d.values()`   |
| `d.items()`     | 所有(键, 值) 二元组（**最常用**） | `for k, v in d.items()` |

## 34. update() — 合并字典

将另一个字典合并到当前字典中。相同的键会被覆盖，新的键会被添加：

```python
person1 = {'name': '王大锤', 'age': 55, 'height': 178}
person2 = {'age': 25, 'addr': '成都市武侯区科华北路62号'}

person1.update(person2)
print(person1)
# {'name': '王大锤', 'age': 25, 'height': 178, 'addr': '成都市武侯区科华北路62号'}
# ↑ age 被 person2 的值覆盖了，addr 是新增的
```

Python 3.9+ 还可以用 `|=` 运算符实现同样的效果：

```python
person1 = {'name': '王大锤', 'age': 55, 'height': 178}
person2 = {'age': 25, 'addr': '成都市武侯区...'}

person1 |= person2
print(person1)
# 结果与 update() 完全相同
```

## 35. 四种删除方式对比

### ① pop(键) — 按键删除（**有返回值**）

删除并返回被删掉的值。键不存在会报 KeyError。

```python
person = {'name': '王大锤', 'age': 25, 'height': 178, 'addr': '成都'}

val = person.pop('age')
print(val)      # 25（被删的值）
print(person)   # {'name': '王大锤', 'height': 178, 'addr': '成都'}
```

### ② popitem() — 删除最后一个（**返回二元组**）

不需要指定键，自动删除并返回最后一个插入的键值对（Python 3.7+ 保持插入顺序）：

```python
person = {'name': '王大锤', 'age': 25, 'addr': '成都'}
item = person.popitem()
print(item)     # ('addr', '成都') ← 被删的键和值
print(person)   # {'name': '王大锤', 'age': 25}
```

### ③ del 语句（**无返回值**）

```python
person = {'name': '王大锤', 'age': 25, 'addr': '成都'}
del person['age']
del person['addr']
print(person)  # {'name': '王大锤'}
```

### ④ clear() — 清空全部

```python
person.clear()
print(person)  # {}
```

### 四种删除方式对比表

| 方法             | 按什么删？ | 有返回值？       | 键不存在时       |
|----------------|----------|--------------|---------------|
| `pop(键)`       | 按**键**删  | ✅ 返回被删的值   | ❌ KeyError   |
| `popitem()`     | 删**最后**一个 | ✅ 返回(键,值)元组 | 不适用（不用传键）  |
| `del dict[键]`   | 按**键**删  | ❌ 无返回值      | ❌ KeyError   |
| `dict.clear()`  | 全部清空    | ❌ 无返回值      | 不适用           |

## 36. 字典嵌套 — 值可以是任何类型

字典的值可以是列表、字典等任意类型，实现复杂数据建模：

```python
person = {
    'name': '王大锤',
    'age': 55,
    'addresses': ['成都市武侯区...', '北京市西城区...'],  # 值是列表
    'car': {                                              # 值是字典（嵌套）
        'brand': 'BMW X7',
        'maxSpeed': 250,
        'length': 5170,
    }
}

# 取嵌套的数据
print(person['addresses'][0])        # 成都市武侯区...
print(person['car']['brand'])        # BMW X7
print(person['car']['maxSpeed'])     # 250
```

## 37. 字典实战案例

### 案例 1：统计字母出现次数

输入一段英文，统计每个字母出现的次数并从高到低排序输出：

```python
sentence = input('请输入一段话: ')
counter = {}

for ch in sentence:
    if 'A' <= ch <= 'Z' or 'a' <= ch <= 'z':
        counter[ch] = counter.get(ch, 0) + 1

sorted_keys = sorted(counter, key=counter.get, reverse=True)

for key in sorted_keys:
    print(f'{key} 出现了 {counter[key]} 次.')
```

**输入：**
```
Man is distinguished, not only by his reason...
```

**输出：**
```
e 出现了 27 次.
n 出现了 21 次.
a 出现了 18 次.
i 出现了 18 次.
s 出现了 16 次.
...
```

**核心技巧**：`counter[ch] = counter.get(ch, 0) + 1`
- 第一次遇到某字母：`get(ch, 0)` 返回 0，加后变成 1
- 之后每次遇到：取出当前次数 +1

### 案例 2：字典生成式筛选股票

找出股价大于 100 元的股票，创建新字典：

```python
stocks = {
    'AAPL': 191.88,
    'GOOG': 1186.96,
    'IBM': 149.24,
    'ORCL': 48.44,
    'ACN': 166.89,
    'FB': 208.09,
    'SYMC': 21.29,
}

# 用字典生成式 + 条件筛选
expensive = {code: price for code, price in stocks.items() if price > 100}

print(expensive)
# {'AAPL': 191.88, 'GOOG': 1186.96, 'IBM': 149.24, 'ACN': 166.89, 'FB': 208.09}
```

## 38. 字典方法速查总表

| 方法/操作                   | 功能              | 改变原字典？ | 有返回值？  |
|--------------------------|-----------------|:----------:|:--------:|
| `len(dict)`               | 获取键值对数量        | ❌         | ✅ 数字     |
| `键 in dict`              | 判断键是否存在        | ❌         | ✅ True/False |
| `dict[键]` / `dict[键] = 值`    | 索引取值 / 修改 / 新增  | ✅         | 取值时有     |
| `dict.get(键, 默认值)`        | 安全取值（键不存在不报错） | ❌         | ✅ 值/None  |
| `dict.keys()`             | 获取所有键          | ❌         | ✅ 视图对象   |
| `dict.values()`           | 获取所有值          | ❌         | ✅ 视图对象   |
| `dict.items()`            | 获取所有键值对        | ❌         | ✅ 视图对象   |
| `dict.update(other)`      | 合并字典            | ✅         | ❌        |
| `dict.pop(键)`            | 弹出指定键的值        | ✅         | ✅ 被删的值   |
| `dict.popitem()`          | 弹出最后一组键值对     | ✅         | ✅ (键,值)  |
| `del dict[键]`            | 按键删除           | ✅         | ❌        |
| `dict.clear()`            | 清空全部           | ✅         | ❌        |
| `dict.copy()`             | 浅拷贝             | ❌         | ✅ 新字典    |
| `dict1 \| dict2` (3.9+)    | 合并（返回新字典）      | ❌         | ✅ 新字典    |

## 今日总结

- **字典核心思想**：用**键**映射到**值**，比列表/元组的下标更语义化——取数据不需要记住位置
- **创建字典**：字面量 `{}` 最常用；`zip()` 配对两个序列；**字典生成式**批量创建+条件筛选最强
- **安全取值**：`get()` 是最重要的方法，键不存在时不报错而是返回默认值——比 `[]` 安全得多
- **遍历字典**：`items()` 最常用，同时拿到键和值；`keys()` 只遍历键
- **合并字典**：`update()` 原地修改；`|=` （Python 3.9+）更简洁
- **删除**：`pop()` 有返回值适合需要被删值的场景；`clear()` 一键清空；`del` 最简单但无返回值
- **嵌套**：值可以是列表、字典等任意类型，`person['car']['brand']` 这样链式访问
- **实战模式**：`counter.get(ch, 0) + 1` 是计数经典写法；字典生成式 + 条件筛选一行搞定过滤
