## 一、面向对象编程入门（DAY19）

### 1. 编程范式

**编程范式**（programming paradigm）= 程序设计的方法论。

| 范式 | 核心思想 |
|---|---|
| 指令式编程 | 程序是指令的集合，CPU 逐条执行 |
| 函数式编程 | 把功能封装成函数，组合函数解决问题 |
| **面向对象编程** | 数据和操作数据的函数封装成**对象**，通过消息传递协作 |

> 面向对象编程更符合人类正常思维习惯，复杂系统用面向对象更易维护。

---

### 2. 类与对象

| 概念 | 说明 |
|---|---|
| **类（class）** | 抽象概念，是对象的蓝图和模板 |
| **对象（object）** | 具体存在，是类的实例，可以接收消息 |

核心口诀：**一切皆为对象，对象都有属性和行为，对象一定属于某个类。**

---

### 3. 定义类

```python
class Student:

    def study(self, course_name):
        print(f'学生正在学习{course_name}.')

    def play(self):
        print(f'学生正在玩游戏.')
```

- 类名首字母大写（驼峰命名）
- 类中的函数叫**方法**，第一个参数固定为 `self`，代表接收消息的对象本身

---

### 4. 创建和使用对象

```python
stu1 = Student()   # 构造器语法：类名()
stu2 = Student()
```

调用方法的两种方式：

```python
# 方式一：类.方法（需手动传入对象）
Student.study(stu1, 'Python程序设计')

# 方式二：对象.方法（推荐，自动传入self）
stu1.study('Python程序设计')
```

> `stu3 = stu2` 不会创建新对象，只是让两个变量指向同一内存地址。

### 8. 案例：数字时钟

```python
import time

class Clock:
    def __init__(self, hour=0, minute=0, second=0):
        self.hour = hour
        self.min = minute
        self.sec = second

    def run(self):
        """走字"""
        self.sec += 1
        if self.sec == 60:
            self.sec = 0
            self.min += 1
            if self.min == 60:
                self.min = 0
                self.hour += 1
                if self.hour == 24:
                    self.hour = 0

    def show(self):
        return f'{self.hour:0>2d}:{self.min:0>2d}:{self.sec:0>2d}'

clock = Clock(23, 59, 58)
while True:
    print(clock.show())
    time.sleep(1)
    clock.run()
```

### 9. 案例：平面上的点

```python
class Point:
    def __init__(self, x=0, y=0):
        self.x, self.y = x, y

    def distance_to(self, other):
        dx = self.x - other.x
        dy = self.y - other.y
        return (dx ** 2 + dy ** 2) ** 0.5

    def __str__(self):
        return f'({self.x}, {self.y})'

p1 = Point(3, 5)
p2 = Point(6, 9)
print(p1.distance_to(p2))  # 5.0
```

## 二、面向对象编程进阶（DAY20）

### 1. 魔术方法（Magic Methods）

> Python 中 `__xx__()` 格式的函数叫魔术方法，在特定条件下**自动调用**。

| 魔术方法 | 触发时机 |
|---|---|
| `__init__` | 创建对象时自动调用，用于初始化属性 |
| `__str__` | `print(对象)` 时调用，返回可读字符串（若未定义则打印内存地址） |
| `__del__` | 对象被销毁时自动调用 |
| `__repr__` | 交互式解释器显示对象时调用 |
| `__lt__` | 使用 `<` 比较时调用 |

---

### 2. 属性可见性

Python 通过命名约定控制访问权限：

| 写法 | 可见性 |
|---|---|
| `name` | 公开（public），外部可访问 |
| `_name` | 受保护（protected），约定不直接访问 |
| `__name` | 私有（private），外部无法直接访问 |

```python
class Student:
    def __init__(self, name, age):
        self.__name = name   # 私有属性
        self.__age = age

stu = Student('王大锤', 20)
# print(stu.__name)  # AttributeError！
# 强制访问（不推荐）：stu._Student__name
```

> Python 哲学："We are all consenting adults here"，开放比封闭好，私有化非必须。

---

### 3. 动态属性

Python 是动态语言，可在运行时给对象动态添加属性：

```python
stu = Student('王大锤', 20)
stu.sex = '男'  # 动态添加属性
```

用 `__slots__` 限制可添加的属性：

```python
class Student:
    __slots__ = ('name', 'age')  # 只允许 name 和 age

    def __init__(self, name, age):
        self.name = name
        self.age = age

stu = Student('王大锤', 20)
stu.sex = '男'  # AttributeError！
```

### 4. 静态方法与类方法

| 方法类型 | 装饰器 | 第一个参数 | 使用场景 |
|---|---|---|---|
| 对象方法 | 无 | `self`（对象） | 操作对象属性 |
| 类方法 | `@classmethod` | `cls`（类） | 操作类本身 |
| 静态方法 | `@staticmethod` | 无 | 与类相关但不依赖对象/类 |

三者都可用 `类名.方法名()` 调用。

```python
class Triangle:
    def __init__(self, a, b, c):
        self.a, self.b, self.c = a, b, c

    @staticmethod
    def is_valid(a, b, c):
        """验证三边是否构成三角形（调用时对象还未创建）"""
        return a + b > c and b + c > a and a + c > b

    @property
    def perimeter(self):
        return self.a + self.b + self.c

    @property
    def area(self):
        p = self.perimeter / 2
        return (p * (p - self.a) * (p - self.b) * (p - self.c)) ** 0.5

if Triangle.is_valid(3, 4, 5):
    t = Triangle(3, 4, 5)
    print(t.perimeter)  # 12，像属性一样访问（@property）
    print(t.area)       # 6.0
```

> `@property` 装饰器：把方法变成属性，用 `对象.属性名` 访问，无需加 `()`。

### 5. 继承

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def eat(self):
        print(f'{self.name}正在吃饭.')


class Student(Person):           # Student 继承 Person
    def __init__(self, name, age):
        super().__init__(name, age)   # 调用父类初始化方法

    def study(self, course_name):
        print(f'{self.name}正在学习{course_name}.')


class Teacher(Person):
    def __init__(self, name, age, title):
        super().__init__(name, age)
        self.title = title

    def teach(self, course_name):
        print(f'{self.name}{self.title}正在讲授{course_name}.')
```

| 术语 | 说明 |
|---|---|
| 父类/超类/基类 | 提供继承信息的类 |
| 子类/派生类 | 接收继承信息的类 |
| `super()` | 获取父类对象，用于调用父类方法 |

类间关系：
- **is-a**（继承）：Student is-a Person
- **has-a**（关联）：Poker has-a Card
- **use-a**（依赖）：Player use-a Card

### 6. 多态

子类重写父类方法后，相同方法调用产生不同行为：

```python
class Employee(metaclass=ABCMeta):
    @abstractmethod
    def get_salary(self):
        pass

class Manager(Employee):
    def get_salary(self): return 15000.0

class Programmer(Employee):
    def __init__(self, name, working_hour=0):
        super().__init__(name)
        self.working_hour = working_hour
    def get_salary(self): return 200 * self.working_hour

class Salesman(Employee):
    def __init__(self, name, sales=0):
        super().__init__(name)
        self.sales = sales
    def get_salary(self): return 1800 + self.sales * 0.05
```

> `isinstance(obj, 类)` 判断对象是否属于某类（包括子类）；比 `type()` 更灵活。

## 三、面向对象编程应用（DAY21）

### 1. 扑克游戏

**枚举类型**（用于有限固定选项）：

```python
from enum import Enum

class Suite(Enum):
    SPADE, HEART, CLUB, DIAMOND = range(4)
```

**运算符重载**（让对象支持 `<` 比较）：

```python
def __lt__(self, other):
    if self.suite == other.suite:
        return self.face < other.face
    return self.suite.value < other.suite.value
```

常用魔术方法与运算符对应：

| 魔术方法 | 运算符 |
|---|---|
| `__lt__` | `<` |
| `__gt__` | `>` |
| `__le__` | `<=` |
| `__ge__` | `>=` |
| `__eq__` | `==` |
| `__ne__` | `!=` |

---

### 2. 工资结算系统（抽象类 + 多态）

```python
from abc import ABCMeta, abstractmethod

class Employee(metaclass=ABCMeta):
    def __init__(self, name):
        self.name = name

    @abstractmethod
    def get_salary(self):
        """抽象方法：只声明，不实现，强制子类重写"""
        pass
```

- `ABCMeta`：抽象类的元类（来自 `abc` 模块）
- `@abstractmethod`：声明抽象方法，子类必须重写，否则无法实例化

## 四、总结

| 概念 | 一句话 |
|---|---|
| 类 | 对象的模板，抽象的蓝图 |
| 对象 | 类的实例，可接收消息 |
| 封装 | 隐藏细节，暴露接口 |
| 继承 | 从父类派生子类，复用代码 |
| 多态 | 同一方法，不同子类行为不同 |
| `__init__` | 初始化方法，创建对象自动调用 |
| `__str__` | print 对象时调用 |
| `@property` | 方法伪装成属性访问 |
| `@staticmethod` | 静态方法，无需 self/cls |
| `@classmethod` | 类方法，第一参数为 cls |
| `__slots__` | 限制对象可添加的属性 |
| `ABCMeta` | 定义抽象类，强制子类实现方法 |
| `isinstance()` | 模糊类型判断，支持继承链 |
| 运算符重载 | 通过魔术方法让对象支持运算符 |
