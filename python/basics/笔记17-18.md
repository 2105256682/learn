## 1. IO 流（Stream）概念

通过"流"的形式允许计算机程序使用相同的方式来访问不同的输入/输出源。stream 是从起源（source）到接收的（sink）的有序数据。把输入/输出源比作"水桶"，那么流就是"管道"。

**文件流**：就是源或者目标都是文件的流。

| 概念 | 说明 |
|------|------|
| Source（源） | 数据的来源，如文件、键盘、网络 |
| Sink（接收） | 数据的目的地，如文件、屏幕、网络 |
| Stream（流） | 连接 Source 和 Sink 的数据通道 |
| 文件流 | 源或目标是文件的 Stream |

## 2. open() — 打开文件流

```python
# 语法
文件对象 = open(目标文件, 访问模式)

# 基本示例（手动关闭）
wf = open('test.txt', 'r')
wf.read()
wf.close()
```

| 参数 | 说明 |
|------|------|
| `目标文件` | 相对路径或绝对路径，字符串类型 |
| `访问模式` | 打开模式，默认 `'r'`（只读） |
| `encoding` | Windows 常用 `'gbk'`，跨平台推荐 `'utf-8'` |

## 3. 访问模式完整表

| 模式 | 描述 |
|------|------|
| `r` | 以只读方式打开文件。文件的指针将会放在文件的开头。**这是默认模式。** |
| `rb` | 以二进制格式打开一个文件用于只读。文件指针将会放在文件的开头。这是默认模式。 |
| `r+` | 打开一个文件用于读写。文件指针将会放在文件的开头。 |
| `rb+` | 以二进制格式打开一个文件用于读写。文件指针将会放在文件的开头。 |
| `w` | 打开一个文件只用于写入。如果该文件已存在则打开文件，并从开头开始编辑，即原有内容会被删除。如果该文件不存在，创建新文件。 |
| `wb` | 以二进制格式打开一个文件只用于写入。如果该文件已存在则打开文件，并从开头开始编辑，即原有内容会被删除。如果该文件不存在，创建新文件。 |
| `w+` | 打开一个文件用于读写。如果该文件已存在则打开文件，并从开头开始编辑，即原有内容会被删除。如果该文件不存在，创建新文件。 |
| `wb+` | 以二进制格式打开一个文件用于读写。如果该文件已存在则打开文件，并从开头开始编辑，即原有内容会被删除。如果该文件不存在，创建新文件。 |
| `a` | 打开一个文件用于追加。如果该文件已存在，文件指针将会放在文件的结尾。新的内容将会被写入到已有内容之后。如果该文件不存在，创建新文件进行写入。 |
| `ab` | 以二进制格式打开一个文件用于追加。如果该文件已存在，文件指针将会放在文件的结尾。新的内容将会被写入到已有内容之后。如果该文件不存在，创建新文件进行写入。 |
| `a+` | 打开一个文件用于读写。如果该文件已存在，文件指针将会放在文件的结尾。文件打开时会是追加模式。如果该文件不存在，创建新文件用于读写。 |
| `ab+` | 以二进制格式打开一个文件用于追加。如果该文件已存在，文件指针将会放在文件的结尾。如果该文件不存在，创建新文件用于读写。 |

## 4. 文件流的读操作

```python
# 打开文件流
wf = open('test.txt', 'r', encoding='gbk')

# read()：默认读取整个文件，或者可以读取指定大小的数据
content = wf.read()        # 读取全部内容，返回字符串
content = wf.read(10)      # 读取 10 个字符

# readlines()：读取所有行，返回列表，每行是一个元素（含 \n）
lines = wf.readlines()     # ['第一行\n', '第二行\n', '第三行']

# readline()：每次读取一行，返回字符串（含末尾换行符）
line = wf.readline()

wf.close()
```

| 方法 | 返回类型 | 特点 |
|------|---------|------|
| `read()` | 字符串 | 一次读完，文件大时占内存 |
| `read(n)` | 字符串 | 读取指定字节数 |
| `readlines()` | 列表 | 每行一个元素，含 `\n` |
| `readline()` | 字符串 | 逐行读，适合超大文件 |

## 5. 文件流的写操作

```python
# w 模式：清空后写入
wf = open('output.txt', 'w', encoding='gbk')
wf.write('Hello, Python!\n')   # write() 写字符串，不会自动换行
wf.write('第二行内容\n')
wf.close()

# a 模式：追加写入（不清空原有内容）
wf = open('output.txt', 'a', encoding='gbk')
wf.write('追加的内容\n')
wf.close()
```

⚠️ **`w` 模式会直接清空原文件！追加用 `a` 模式。**

## 6. seek() 指针操作

文件有一个隐式"指针"记录当前读写位置，`seek()` 可以移动该指针。

```python
# 语法
文件对象.seek(偏移量, 起始位置)
# 起始位置：0 = 文件开头，1 = 当前位置，2 = 文件结尾位置

# tell() 函数返回当前指针位置
文件对象.tell()
```

```python
# 实战案例：在第一个 hello 的后面插入 world
wf = open('test2.txt', 'r+', encoding='gbk')  # 在指定位置写入数据，会造成覆盖

# 把指针移动到第一个 hello 的后面（第 5 个字节）
wf.seek(5, 0)

# 把第一个 hello 后面的内容先读取出来
after = wf.read()      # 读完之后，指针又到了文件的末尾

# 把指针移动回第一个 hello 的后面
wf.seek(5, 0)
wf.write('world' + after)
wf.close()
```

| 参数 | 值 | 含义 |
|------|---|------|
| 起始位置 | `0` | 文件开头（最常用） |
| 起始位置 | `1` | 当前位置 |
| 起始位置 | `2` | 文件结尾位置 |

## 7. close() — 关闭文件流

```python
wf.close()   # 释放文件资源，必须调用，否则内容可能不会写入磁盘
```

### with 语句 — 推荐写法（自动关闭）

```python
# ✅ 推荐：with 块结束后自动调用 close()，无需手动关闭
with open('hello.txt', 'r', encoding='utf-8') as f:
    content = f.read()
    print(content)
```

## 8. 文件编码问题

```python
# Windows 系统默认编码是 GBK，打开中文文件建议指定编码
with open('data.txt', 'r', encoding='gbk') as f:
    content = f.read()

# 跨平台推荐 utf-8
with open('data.txt', 'r', encoding='utf-8') as f:
    content = f.read()

# errors 参数处理编码错误
with open('data.txt', 'r', encoding='utf-8', errors='ignore') as f:
    content = f.read()   # 忽略无法解码的字符
```

| 常见编码 | 说明 |
|---------|------|
| `utf-8` | 推荐，支持全球字符 |
| `gbk` / `gb2312` | 中文 Windows 旧编码 |
| `ascii` | 仅英文，范围最窄 |

## 9. os 模块 — 文件与目录操作（完整 21 个）

```python
import os                                  # 使用之前先导入
```

## 11. shutil 模块 — 高级文件操作

```python
import shutil

shutil.copy('src.txt', 'dst.txt')          # 复制文件（不保留元数据）
shutil.copy2('src.txt', 'dst.txt')         # 复制文件（保留时间戳等元数据，推荐）
shutil.copytree('src_folder', 'dst_folder') # 递归复制整个文件夹（dst 不能已存在）
shutil.move('old_path', 'new_path')         # 移动文件或文件夹（跨盘也可用）
shutil.rmtree('folder_to_delete')           # ⚠️ 删除整个文件夹（含所有内容，不可恢复！）
```

| 函数 | 作用 | 注意 |
|------|------|------|
| `shutil.copy(src, dst)` | 复制文件 | 不保留元数据 |
| `shutil.copy2(src, dst)` | 复制文件 | 保留元数据（推荐） |
| `shutil.copytree(src, dst)` | 递归复制文件夹 | dst 不能已存在 |
| `shutil.move(src, dst)` | 移动/重命名 | 跨盘符也能用 |
| `shutil.rmtree(path)` | 删除整个文件夹 | ⚠️ 不可恢复，慎用 |

## 12. pathlib 模块 — 面向对象路径操作（推荐）

Python 3.4+ 提供的现代路径库，用 `Path` 对象代替字符串拼接，更简洁安全。

```python
from pathlib import Path

p = Path('D:/projects/data/file.txt')

print(p.name)       # file.txt（文件名含扩展名）
print(p.stem)       # file（文件名不含扩展名）
print(p.suffix)     # .txt（扩展名）
print(p.parent)     # D:/projects/data（父目录）

# 路径拼接（用 / 运算符）
full = Path('D:/projects') / 'data' / 'file.txt'

# 判断
print(p.exists())   # 是否存在
print(p.is_file())  # 是否是文件
print(p.is_dir())   # 是否是文件夹

# 读写文件
Path('output.txt').write_text('Hello!', encoding='utf-8')
content = Path('output.txt').read_text(encoding='utf-8')

# 创建多层目录
Path('a/b/c').mkdir(parents=True, exist_ok=True)

# 遍历目录
for item in Path('D:/projects').iterdir():
    print(item)

# 通配符查找（** 表示递归）
for f in Path('D:/projects').glob('**/*.txt'):
    print(f)
```

```python
import os
```

## 14. 实战案例 — 统计目录所有文件大小

```python
import os

def get_folder_size(folder_path):
    """递归统计文件夹总大小（字节）"""
    total = 0
    for root, dirs, files in os.walk(folder_path):
        for file in files:
            file_path = os.path.join(root, file)
            total += os.path.getsize(file_path)
    return total

size = get_folder_size('D:/projects')
print(f'总大小：{size / 1024 / 1024:.2f} MB')
```

## 今日总结

- **IO 流**：程序通过"流"访问文件，文件流就是源或目标是文件的流
- **open() 三要素**：目标文件 + 访问模式 + encoding（Windows 用 gbk，跨平台用 utf-8）
- **12 种访问模式**：r/rb/r+/rb+（只读）、w/wb/w+/wb+（写入覆盖）、a/ab/a+/ab+（追加）
- **读操作三选一**：`read()`（全读）/ `readlines()`（列表）/ `readline()`（逐行）
- **seek() 指针**：`seek(偏移量, 0/1/2)` 移动指针；`tell()` 查当前位置
- **关闭**：手动 `close()` 或用 `with` 语句自动关闭（推荐 with）
- **os 模块 21 个函数**：覆盖路径判断、目录创建/删除、文件重命名、遍历目录树等全部操作
- **shutil**：复制/移动/整个目录删除，`rmtree` 不可恢复慎用
- **pathlib**：Python 3.4+ 推荐，`Path` 对象 + `/` 拼接，内置读写方法
