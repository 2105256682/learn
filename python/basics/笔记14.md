## 39. 集合（Set）简介

集合是 Python 中的容器型数据类型，用花括号 `{}` 表示，但与字典不同，集合里只有值，没有键。

集合的三大特性（数学定义）：

| 特性   | 含义                             |
|------|--------------------------------|
| **无序性** | 元素没有固定顺序，不能用下标访问                 |
| **互异性** | 元素不能重复，重复的值自动去掉                  |
| **确定性** | 一个元素要么在集合里，要么不在，没有模糊状态           |

```python
set1 = {1, 2, 3, 3, 3, 2}
print(set1)  # {1, 2, 3} ← 重复元素自动去除
```

> ⚠️ **注意：集合不支持索引运算！**（不能 `set1[0]`，因为元素无序）

## 40. 集合的 4 种创建方式

### 方式 1：字面量 `{}`（至少一个元素）

```python
set1 = {1, 2, 3, 3, 3, 2}
print(set1)  # {1, 2, 3}

set2 = {'banana', 'apple', 'apple', 'grape'}
print(set2)  # {'banana', 'apple', 'grape'} （顺序随机）
```

> ⚠️ **`{}` 是空字典，不是空集合！空集合必须用 `set()`**

### 方式 2：`set()` 构造器（可将其他序列转为集合）

```python
# 字符串转集合（自动去重）
set3 = set('hello')
print(set3)  # {'h', 'e', 'l', 'o'} ← 重复的 l 只保留一个

# 列表转集合（去重神器）
set4 = set([1, 2, 2, 3, 3, 3, 2, 1])
print(set4)  # {1, 2, 3}

# 创建空集合
empty = set()
print(type(empty))  # <class 'set'>
```

### 方式 3：集合生成式（带条件过滤）

```python
# 1~19 中能被 3 或 7 整除的数
set5 = {num for num in range(1, 20) if num % 3 == 0 or num % 7 == 0}
print(set5)  # {3, 6, 7, 9, 12, 14, 15, 18}
```

### 方式 4：`frozenset()` — 不可变集合

```python
fset = frozenset({1, 3, 5, 7})
print(fset)  # frozenset({1, 3, 5, 7})
# 可以当其他集合的元素（因为是不可变类型）
```

### 4 种方式对比

| 方式             | 写法                        | 适用场景                  |
|----------------|-----------------------------|-------------------------|
| 字面量 `{}`       | `{1, 2, 3}`                  | 已知具体元素               |
| `set(序列)`      | `set([1,2,2,3])`              | 把其他序列去重转换为集合       |
| 集合生成式         | `{x for x in ... if ...}`    | 按规则批量生成 + 条件过滤      |
| `frozenset()`   | `frozenset({1,2,3})`         | 需要不可变集合（可作为其他集合元素）|

## 41. 元素必须是 hashable 类型

集合底层用哈希存储，所以**元素必须是不可变类型（hashable）**：

| 可以放入集合 ✅ | 不能放入集合 ❌ |
|------------|------------|
| `int`、`float`、`bool` | `list`（列表） |
| `str`（字符串）           | `set`（集合本身）|
| `tuple`（元组）           | `dict`（字典）  |

```python
# 正确
s = {1, 3.14, True, 'hello', (1, 2)}

# 错误！列表不能作为集合元素
# s = {[1, 2], [3, 4]}  # TypeError: unhashable type: 'list'
```

## 42. 元素遍历

集合不支持索引，只能用 `for-in` 遍历，且**每次输出顺序可能不同**：

```python
set1 = {'Python', 'C++', 'Java', 'Kotlin', 'Swift'}

print(len(set1))  # 5（用 len 获取元素个数）

for elem in set1:
    print(elem)
# 输出顺序随机，体现集合的无序性
```

## 43. 成员运算 `in`

判断元素是否在集合中，**性能比列表快得多**（底层哈希查找）：

```python
set1 = {11, 12, 13, 14, 15}
print(10 in set1)       # False
print(15 in set1)       # True
print(15 not in set1)   # False

set2 = {'Python', 'Java', 'C++', 'Swift'}
print('Ruby' in set2)   # False
print('Java' in set2)   # True
```

> 记忆口诀：**集合查元素用 `in`，比列表快（O(1) vs O(n)）**

## 44. 四种二元运算（重点！）

两个集合之间可以做交集、并集、差集、对称差：

```python
set1 = {1, 2, 3, 4, 5, 6, 7}
set2 = {2, 4, 6, 8, 10}

# 交集：两个集合共有的元素
print(set1 & set2)                      # {2, 4, 6}
print(set1.intersection(set2))          # {2, 4, 6}

# 并集：两个集合合并（去重）
print(set1 | set2)                      # {1, 2, 3, 4, 5, 6, 7, 8, 10}
print(set1.union(set2))                 # {1, 2, 3, 4, 5, 6, 7, 8, 10}

# 差集：set1 有但 set2 没有的
print(set1 - set2)                      # {1, 3, 5, 7}
print(set1.difference(set2))            # {1, 3, 5, 7}

# 对称差：两个集合各自独有的（不共有的）
print(set1 ^ set2)                      # {1, 3, 5, 7, 8, 10}
print(set1.symmetric_difference(set2))  # {1, 3, 5, 7, 8, 10}
```

### 四种运算总结表

| 运算    | 运算符 | 方法                         | 含义                        |
|-------|-----|------------------------------|-----------------------------|
| 交集    | `&`  | `.intersection()`            | 两集合共有的元素              |
| 并集    | `\|`  | `.union()`                   | 两集合全部元素（去重）          |
| 差集    | `-`  | `.difference()`              | 左边有、右边没有的元素          |
| 对称差  | `^`  | `.symmetric_difference()`    | 两边各自独有的（排除共有的）    |

### 复合赋值运算

```python
set1 = {1, 3, 5, 7}
set2 = {2, 4, 6}

set1 |= set2          # 等价于 set1 = set1 | set2
# set1.update(set2)   # 同效果
print(set1)  # {1, 2, 3, 4, 5, 6, 7}

set3 = {3, 6, 9}
set1 &= set3          # 等价于 set1 = set1 & set3
print(set1)  # {3, 6}

set2 -= set1          # 等价于 set2 = set2 - set1
print(set2)  # {2, 4}
```

## 45. 比较运算（子集、超集）

```python
set1 = {1, 3, 5}
set2 = {1, 2, 3, 4, 5}
set3 = {5, 4, 3, 2, 1}  # 与 set2 元素相同

# 相等判断
print(set2 == set3)  # True（元素完全相同就相等，顺序不影响）

# 真子集（set1 是 set2 的真子集）
print(set1 < set2)   # True
print(set1 <= set2)  # True（子集，包含相等）

# 超集
print(set2 > set1)   # True

# 等价方法写法
print(set1.issubset(set2))    # True（set1 是 set2 的子集）
print(set2.issuperset(set1))  # True（set2 是 set1 的超集）
```

### 比较运算总结表

| 运算符 | 方法                  | 含义              |
|------|---------------------|-----------------|
| `==` | —                   | 两集合元素完全相同       |
| `<`  | —                   | 真子集（左边 ⊊ 右边）    |
| `<=` | `.issubset()`        | 子集（左边 ⊆ 右边）     |
| `>`  | —                   | 真超集（左边 ⊋ 右边）   |
| `>=` | `.issuperset()`      | 超集（左边 ⊇ 右边）    |

## 46. 集合的增删方法

```python
set1 = {1, 10, 100}

# 添加单个元素
set1.add(1000)
set1.add(10000)
print(set1)  # {1, 10, 100, 1000, 10000}（顺序随机）
```

### 三种删除方式

```python
set1 = {1, 10, 100, 1000}

# discard(元素)：删除元素，元素不存在时不报错（安全）
set1.discard(10)
set1.discard(999)  # 不存在也不报错
print(set1)  # {1, 100, 1000}

# remove(元素)：删除元素，元素不存在时报 KeyError
if 100 in set1:
    set1.remove(100)
print(set1)  # {1, 1000}

# pop()：随机删除一个元素，并返回被删的值
val = set1.pop()
print(val)   # 被删的值（随机）

# clear()：清空全部
set1.clear()
print(set1)  # set()
```

### 三种删除方式对比

| 方法          | 按什么删？  | 有返回值？    | 元素不存在时       |
|-------------|---------|:----------:|-----------------|
| `discard(x)` | 按**值**删  | ❌ 无返回值   | ✅ 安全，不报错       |
| `remove(x)`  | 按**值**删  | ❌ 无返回值   | ❌ **KeyError**  |
| `pop()`      | **随机**删  | ✅ 返回被删的值 | 集合为空时 KeyError |

> 记忆口诀：**不确定元素在不在就用 `discard`，确定在再用 `remove`**

## 47. isdisjoint() — 判断两集合是否无交集

```python
set1 = {'Java', 'Python', 'C++', 'Kotlin'}
set2 = {'Kotlin', 'Swift', 'Java', 'Dart'}
set3 = {'HTML', 'CSS', 'JavaScript'}

print(set1.isdisjoint(set2))  # False（有共同元素：Java、Kotlin）
print(set1.isdisjoint(set3))  # True（没有任何共同元素）
```

## 48. frozenset — 不可变集合

`frozenset` 相当于集合版的元组：元素不能增删，但支持所有集合运算，且可以作为其他集合的元素。

```python
fset1 = frozenset({1, 3, 5, 7})
fset2 = frozenset(range(1, 6))  # frozenset({1, 2, 3, 4, 5})

print(fset1 & fset2)  # frozenset({1, 3, 5})
print(fset1 | fset2)  # frozenset({1, 2, 3, 4, 5, 7})
print(fset1 - fset2)  # frozenset({7})
print(fset1 < fset2)  # False
```

### set vs frozenset 对比

| 比较项   | `set`      | `frozenset` |
|---------|-----------|-------------|
| 是否可变  | ✅ 可变     | ❌ 不可变      |
| 能否去重  | ✅         | ✅           |
| 支持集合运算 | ✅       | ✅           |
| 能否作为集合元素 | ❌    | ✅           |
| 类比     | 像 `list`  | 像 `tuple`   |

## 今日总结

- **集合三大特性**：无序（不能索引）、互异（自动去重）、确定（`in` 判断）
- **创建集合**：`{值, ...}` 字面量最常用；`set(序列)` 是**去重利器**；注意 `{}` 是空字典不是空集合
- **元素限制**：只能放 hashable 类型（int/str/tuple 等）；列表、字典、集合本身不能放进集合
- **二元运算**：`&` 交集、`|` 并集、`-` 差集、`^` 对称差——**这四个是最高频考点**
- **删除元素**：不确定元素在不在就用 `discard`（安全）；`remove` 不存在会崩
- **成员运算性能**：集合的 `in` 比列表快得多，查找元素优先用集合
- **frozenset**：不可变版集合，可以作为其他集合的元素，类似元组之于列表
