## 09. 列表（List）简介

列表是 Python 中**最常用的数据类型**，用方括号 `[]` 表示，可以存放多个数据，每个数据叫一个**元素**，元素有**下标**（从 0 开始）。

```python
languages = ['Python', 'Java', 'C++', 'Kotlin']
print(languages[0])    # 输出：Python（取第1个元素）
print(languages[-1])   # 输出：Kotlin（-1 表示最后一个）
```

| 操作       | 写法                  | 说明               |
|----------|---------------------|------------------|
| 取值       | `list[下标]`          | 下标从 0 开始         |
| 倒数取值     | `list[-1]`           | -1 最后一个，-2 倒数第二个 |
| 切片       | `list[起始:结束]`      | 包含起始，不包含结束      |
| 修改       | `list[下标] = 新值`     | 直接赋值修改元素        |

## 10. 列表遍历 — for 循环

### 方式 1：通过索引遍历（需要下标时）

`len()` 获取长度，`range()` 生成所有下标：

```python
languages = ['Python', 'Java', 'C++', 'Kotlin']
for index in range(len(languages)):
    print(f'下标{index}：{languages[index]}')
```

**输出：**
```
下标0：Python
下标1：Java
下标2：C++
下标3：Kotlin
```

### 方式 2：直接遍历元素（仅需要值时，**推荐**）

循环变量直接接收列表每个元素，更简洁：

```python
language_list = ['Python', 'Java', 'C++', 'Kotlin']
for language in language_list:
    print(language)
```

**输出：**
```
Python
Java
C++
Kotlin
```

| 遍历方式 | 适用场景 | 特点 |
|---------|---------|------|
| `range(len())` + 索引 | **需要用到下标**时 | 能同时拿到位置和值 |
| `for x in list` | **只需要值**时 | 代码简洁，推荐优先使用 |

## 11. len() — 获取列表长度

内置函数，返回列表中元素的个数。

```python
lst = [10, 20, 30]
print(len(lst))  # 输出：3（一共 3 个元素）
```

> 注意：`len()` 是**内置函数**，不是列表的方法，直接调用即可。

---

## 12. index() — 查找数据的下标

返回指定数据在列表中**第一次出现**的下标位置。

```python
lst = [35, 12, 35, 87]
print(lst.index(35))  # 输出：0（35 第一次出现在下标 0）
```

⚠️ **注意：如果数据不存在会报错 `ValueError`**

```python
# 安全写法：先判断再查找
if 99 in lst:
    print(lst.index(99))
else:
    print("不存在")
```

---

## 13. count() — 统计数据出现次数

返回数据在列表中出现的总次数。

```python
lst = [35, 12, 35, 87, 35]
print(lst.count(35))  # 输出：3（数字 35 出现了 3 次）
print(lst.count(12))  # 输出：1
print(lst.count(99))  # 输出：0（不存在就返回 0，不会报错）
```

---

## 14. append() — 尾部追加单个数据

在列表末尾追加一个元素，**原地修改原列表**，没有返回值。

```python
lst = [1, 2, 3]
lst.append(4)
print(lst)  # 输出：[1, 2, 3, 4]
```

⚠️ **坑：append 追加列表时会整个嵌套进去！**

```python
lst = [1, 2, 3]
lst.append([5, 6])
print(lst)  # 输出：[1, 2, 3, [5, 6]] ← 整个列表变成一个元素！
```

---

## 15. extend() — 尾部批量追加（拆开序列）

把传入的序列**拆成单个元素**逐个添加到末尾，不会嵌套。

```python
lst = [1, 2, 3]
lst.extend([4, 5])
print(lst)  # 输出：[1, 2, 3, 4, 5] ← 拆开逐个添加
```

### append vs extend 对比（重点！）

| 方法 | 追加 `[5, 6]` 的结果 | 说明 |
|------|-------------------|------|
| `append([5,6])` | `[1, 2, 3, [5, 6]]` | 整个列表作为 **1 个元素** 嵌套 |
| `extend([5,6])` | `[1, 2, 3, 5, 6]` | 拆开后 **3 个元素** 分别添加 |

> 记忆口诀：**append 是"整体塞"，extend 是"拆开加"**

---

## 16. insert() — 指定位置插入

在指定的下标位置插入数据，原来该位置的元素及之后全部后移。

```python
lst = [1, 2, 3]
lst.insert(1, 99)  # 在下标 1 的位置插入 99
print(lst)  # 输出：[1, 99, 2, 3]

lst.insert(0, 100)  # 在最前面插入
print(lst)  # 输出：[100, 1, 99, 2, 3]
```

## 17. 三种删除方式对比

### ① pop() — 弹出删除（**有返回值**）

删除并返回被删掉的值。不传参默认删最后一个。

```python
lst = [10, 20, 30, 40]

val = lst.pop()       # 不传参 → 删最后一个
print(val)            # 输出：40（被删除的值）
print(lst)            # 输出：[10, 20, 30]

val2 = lst.pop(1)     # 传入下标 → 删下标 1
print(val2)           # 输出：20
print(lst)            # 输出：[10, 30]
```

### ② del 语句 — 按下标/切片删除（**无返回值**）

```python
lst = [10, 20, 30, 40]

del lst[1]            # 删除下标 1
print(lst)            # 输出：[10, 30, 40]

del lst[1:]           # 切片批量删除（从下标 1 到末尾全删）
print(lst)            # 输出：[10]
```

### ③ remove() — 按值删除（**无返回值**）

删除第一个匹配的数据（只删第一个，后面的不管）。

```python
lst = [35, 12, 35, 87]
lst.remove(35)
print(lst)  # 输出：[12, 35, 87] ← 只删了第一个 35，第二个保留
```

⚠️ **注意：数据不存在会报错 `ValueError`**

### 三种删除对比表

| 方法 | 按什么删？ | 有返回值？ | 典型用法 |
|------|-----------|:---------:|---------|
| `pop(下标)` | 按**下标**删 | ✅ 返回被删值 | 需要用被删的数据时 |
| `del lst[下标]` | 按**下标**删 | ❌ 无返回值 | 简单删除不要返回值时 |
| `remove(值)` | 按**值**删 | ❌ 无返回值 | 知道要删的内容但不知道下标 |

## 18. reverse() 和 sort()

### reverse() — 反转

原地翻转列表顺序，无返回值。

```python
lst = [1, 2, 3, 4]
lst.reverse()
print(lst)  # 输出：[4, 3, 2, 1]
```

### sort() — 排序

原地升序排序（数字从小到大），传 `reverse=True` 可降序。

```python
lst = [99, 12, 35, 68]

lst.sort()              # 升序（默认）
print(lst)             # 输出：[12, 35, 68, 99]

lst.sort(reverse=True)  # 降序
print(lst)             # 输出：[99, 68, 35, 12]
```

⚠️ **注意：sort() 直接修改原列表，不产生新列表。想保留原列表先用 `.copy()`**

## 19. 列表方法速查总表

| 方法/函数 | 功能 | 改变原列表？ | 有返回值？ |
|----------|------|:----------:|:---------:|
| `len(list)` | 获取元素个数 | ❌ | ✅ 数字 |
| `list.index(x)` | 查找第一次出现的下标 | ❌ | ✅ 下标数字 |
| `list.count(x)` | 统计出现次数 | ❌ | ✅ 数字 |
| `list.append(x)` | 尾部追加**单个**元素 | ✅ | ❌ |
| `list.extend(seq)` | 尾部**拆开**追加序列 | ✅ | ❌ |
| `list.insert(i, x)` | 指定下标位置插入 | ✅ | ❌ |
| `list.pop(i)` | 弹出删除指定下标 | ✅ | ✅ 被删的值 |
| `del list[i]` | 按下标/切片删除 | ✅ | ❌ |
| `list.remove(x)` | 按值删除第一个匹配 | ✅ | ❌ |
| `list.reverse()` | 原地反转 | ✅ | ❌ |
| `list.sort()` | 原地升序排序 | ✅ | ❌ |

## 今日总结

- **列表遍历**：`for x in list` 最常用（只要值）；`range(len())` 需要下标时用
- **增加元素**：`append` 加单个、`extend` 批量加（拆开）、`insert` 指定位置插
- **删除元素**：`pop` 按下标删且有返回值、`del` 按下标/切片删、`remove` 按值删
- **append vs extend**：前者整体嵌套，后者拆开逐个——这是**最高频的考点和易错点**
- **查找统计**：`index` 找下标（找不到报错）、`count` 数次数（找不到返回 0）
- **排序反转**：`sort` 排序、`reverse` 反转，都是**原地修改**原列表
