## 一、文件读写和异常处理（DAY21）

### 1. 数据持久化与文件系统

- **数据持久化**：将内存中的数据转移到硬盘文件中保存
- **文件系统**：用文件和树形目录的抽象概念代替物理设备的数据块概念

### 2. open 函数与文件模式

```python
file = open('文件名', '模式', encoding='utf-8')
```

| 模式 | 含义 |
|---|---|
| `'r'` | 读取（默认） |
| `'w'` | 写入（先截断原有内容） |
| `'x'` | 写入，文件已存在则报错 |
| `'a'` | 追加（写入到末尾） |
| `'b'` | 二进制模式 |
| `'t'` | 文本模式（默认） |
| `'+'` | 更新（可读可写） |

### 3. 文件读写方法

| 方法 | 说明 |
|---|---|
| `read()` | 读取全部内容 |
| `readlines()` | 按行读取到列表 |
| `for line in file` | 逐行迭代读取 |
| `write()` | 写入内容 |
| `close()` | 关闭文件，释放资源 |

### 4. with 上下文管理器（推荐）

```python
try:
    with open('致橡树.txt', 'r', encoding='utf-8') as file:
        print(file.read())
except FileNotFoundError:
    print('无法打开指定的文件!')
```

> `with` 语句会自动调用 `file.close()`，无需手动关闭，代码更优雅。

### 5. 异常处理五个关键字

| 关键字 | 作用 |
|---|---|
| `try` | 放置可能出错的代码 |
| `except` | 捕获异常并处理 |
| `else` | `try` 无异常时执行 |
| `finally` | 无论是否异常都执行（常用于释放资源） |
| `raise` | 主动抛出异常 |

### 6. 异常继承体系

```
BaseException
 ├── SystemExit
 ├── KeyboardInterrupt
 ├── GeneratorExit
 └── Exception      ← 所有常规异常的父类
      ├── ValueError
      ├── TypeError
      ├── FileNotFoundError
      ├── IndexError / KeyError
      └── ...
```

### 7. 自定义异常

```python
class InputError(ValueError):
    pass

def fac(num):
    if num < 0:
        raise InputError('只能计算非负整数的阶乘')
    return 1 if num in (0, 1) else num * fac(num - 1)
```

### 8. 二进制文件读写

```python
# 复制图片（大文件分块读取）
try:
    with open('guido.jpg', 'rb') as f1, open('吉多.jpg', 'wb') as f2:
        data = f1.read(512)  # 每次读 512 字节
        while data:
            f2.write(data)
            data = f1.read()
except FileNotFoundError:
    print('文件不存在')
```

## 二、对象的序列化和反序列化（DAY22）

### 1. JSON 概述

- **JSON**（JavaScript Object Notation）：跨语言跨平台的数据交换标准，已取代 XML 成为事实标准
- **序列化**：对象 → 字符串/字节（可存储、可传输）
- **反序列化**：字符串/字节 → 对象（恢复使用）

### 2. 数据类型对应

| JSON | Python |
|---|---|
| object | dict |
| array | list |
| string | str |
| number | int / float |
| boolean | bool |
| null | None |

### 3. json 模块四个核心函数

| 函数 | 功能 |
|---|---|
| `dumps(obj)` | 对象 → JSON 字符串 |
| `dump(obj, file)` | 对象 → 写入 JSON 文件 |
| `loads(json_str)` | JSON 字符串 → 对象 |
| `load(file)` | 读取 JSON 文件 → 对象 |

```python
import json

my_dict = {'name': '骆昊', 'age': 40}

# 序列化（中文正常显示）
json_str = json.dumps(my_dict, ensure_ascii=False, indent=2)

# 写入文件
with open('data.json', 'w', encoding='utf-8') as f:
    json.dump(my_dict, f, ensure_ascii=False, indent=2)

# 从文件读取
with open('data.json', 'r', encoding='utf-8') as f:
    data = json.load(f)
```

## 三、Python 读写 CSV 文件（DAY23）

### 1. CSV 特点

- **纯文本**格式，跨平台通用
- 由一条条记录组成（每行一条）
- 字段用**逗号**（或指定分隔符）分隔
- 每条记录字段序列一致

### 2. csv 模块读写

```python
import csv

# 写入 CSV
with open('scores.csv', 'w', newline='', encoding='utf-8') as f:
    writer = csv.writer(f)
    writer.writerow(['姓名', '语文', '数学', '英语'])
    writer.writerow(['关羽', 98, 86, 61])

# 读取 CSV
with open('scores.csv', 'r', encoding='utf-8') as f:
    reader = csv.reader(f)
    for row in reader:
        print(row)
```

### 3. 自定义分隔符和包围方式

```python
writer = csv.writer(f, delimiter='|', quoting=csv.QUOTE_ALL)
```

### 4. pandas 扩展（推荐）

```python
import pandas as pd

# 读取
df = pd.read_csv('scores.csv')

# 写入
df.to_csv('output.csv', index=False)
```

### 2. 模板批量生成（核心技巧）

```python
# 在 Word 模板中用占位符如 {name}
# 遍历 runs 进行替换（不能直接改 p.text，会丢失样式）
for p in doc.paragraphs:
    if '{' not in p.text:
        continue
    for run in p.runs:
        if '{' in run.text:
            run.text = run.text.replace('{name}', '骆昊')
```

## 八、Python 发送邮件和短信（DAY29）

### 1. 发送邮件（SMTP）

```python
import smtplib
from email.header import Header
from email.mime.multipart import MIMEMultipart
from email.mime.text import MIMEText

# 创建邮件对象
email = MIMEMultipart()
email['From'] = '发件人@126.com'
email['To'] = '收件人@qq.com'
email['Subject'] = Header('邮件主题', 'utf-8')

# 正文（plain 纯文本 / html）
email.attach(MIMEText('邮件内容', 'plain', 'utf-8'))

# 发送
smtp = smtplib.SMTP_SSL('smtp.126.com', 465)
smtp.login('发件人@126.com', '授权码')
smtp.sendmail('发件人', ['收件人'], email.as_string())
smtp.quit()
```

### 2. 发送带附件的邮件

```python
from urllib.parse import quote

with open('文件.docx', 'rb') as f:

    attachment['content-type'] = 'application/octet-stream'
    filename = quote('文件.docx')
    attachment['content-disposition'] = f'attachment; filename="{filename}"'
```

## 九、正则表达式的应用（DAY30）

### 1. 正则表达式基本符号

| 符号 | 含义 | 示例 |
|---|---|---|
| `.` | 任意字符 | `b.t` → bat, but, b1t |
| `\w` | 字母/数字/下划线 | `b\wt` → bat, b1t, b_t |
| `\d` | 数字 | `\d\d` → 01, 23 |
| `\s` | 空白字符 | `love\syou` → love you |
| `^` | 字符串开头 | `^The` |
| `$` | 字符串结尾 | `.exe$` |
| `*` | 0次或多次 | `\w*` |
| `+` | 1次或多次 | `\w+` |
| `?` | 0次或1次 | `\w?` |
| `{n}` | 恰好 n 次 | `\d{3}` |
| `{m,n}` | m 到 n 次 | `\w{3,6}` |
| `[]` | 字符集 | `[aeiou]` |
| `\|` | 分支 | `foo\|bar` |
| `(exp)` | 捕获分组 | `(\d+)` |
| `.*?` | 非贪婪匹配 | 尽可能少匹配 |

### 2. re 模块核心函数

| 函数 | 功能 |
|---|---|
| `match(pattern, string)` | 从开头匹配，成功返回匹配对象 |
| `search(pattern, string)` | 搜索第一次出现，成功返回匹配对象 |
| `findall(pattern, string)` | 查找所有匹配，返回列表 |
| `finditer(pattern, string)` | 查找所有匹配，返回迭代器 |
| `sub(pattern, repl, string)` | 替换匹配内容 |
| `split(pattern, string)` | 按正则分割字符串 |
| `compile(pattern)` | 编译正则表达式，重复使用时效率更高 |

### 3. 代码示例

```python
import re

# 验证用户名和 QQ 号
username = input('请输入用户名: ')
qq = input('请输入QQ号: ')
m1 = re.match(r'^[0-9a-zA-Z_]{6,20}$', username)
m2 = re.fullmatch(r'[1-9]\d{4,11}', qq)

# 提取手机号
sentence = '我的手机号是13512346789'
pattern = re.compile(r'(?<=\D)1[34578]\d{9}(?=\D)')
tels = re.findall(pattern, sentence)

# 替换不良内容
sentence = 'Oh, shit! 你是傻逼吗?'
purified = re.sub('fuck|shit|[傻煞沙][比笔逼]', '*', sentence, flags=re.IGNORECASE)

# 拆分字符串
poem = '窗前明月光，疑是地上霜。'
print(re.split(r'[，。]', poem))
```
