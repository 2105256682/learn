## 1. 函数简介与定义

函数是**把一段可以重复使用的代码封装起来**，通过名字调用执行。定义一次，随时调用，避免重复写同样的代码。

```python
# 语法
def 函数名(参数列表):
    """文档字符串（可选）"""
    函数体
    return 返回值  # 可选
```

```python
# 定义一个计算两数之和的函数
def add(a, b):
    """返回 a 和 b 的和"""
    return a + b

# 调用函数
result = add(3, 5)
print(result)  # 输出：8
```

| 关键字 | 说明 |
|--------|------|
| `def` | 定义函数的关键字，必须有 |
| `函数名` | 小写字母+下划线，见名知意 |
| `参数列表` | 可以有 0 个或多个参数，括号不能省 |
| `return` | 返回结果；没有 return 则返回 None |
| `文档字符串` | 用 `"""..."""` 写在第一行，描述函数用途 |

## 2. 参数类型 — 位置参数

调用时**按顺序传入**，顺序和定义时一致，最基础的参数类型。

```python
def greet(name, age):
    print(f'你好，我叫{name}，今年{age}岁')

greet('小明', 18)   # 输出：你好，我叫小明，今年18岁
greet(18, '小明')   # ⚠️ 顺序错了！输出：你好，我叫18，今年小明岁
```

⚠️ **注意：位置参数必须按顺序传入，多传少传都会报 TypeError**

---

## 3. 参数类型 — 关键字参数

调用时**通过参数名=值**传入，顺序可以不一样，代码可读性更强。

```python
def greet(name, age):
    print(f'你好，我叫{name}，今年{age}岁')

# 用关键字参数，顺序随便
greet(age=20, name='小红')   # 输出：你好，我叫小红，今年20岁
greet(name='小蓝', age=22)   # 输出：你好，我叫小蓝，今年22岁
```

> **混合使用规则**：位置参数必须在关键字参数**前面**，否则报 `SyntaxError`

```python
greet('小绿', age=19)   # ✅ 正确：位置参数在前
# greet(name='小绿', 19)  # ❌ 错误：关键字参数后不能跟位置参数
```

## 4. 参数类型 — 默认参数

定义时给参数一个**默认值**，调用时可以不传该参数。

```python
def greet(name, age=18):   # age 有默认值 18
    print(f'你好，我叫{name}，今年{age}岁')

greet('小明')        # 不传 age → 用默认值 18 → 输出：你好，我叫小明，今年18岁
greet('小红', 25)    # 传了 age → 覆盖默认值 → 输出：你好，我叫小红，今年25岁
```

⚠️ **规则：默认参数必须放在普通参数的后面**

```python
def func(a, b=10, c=20):   # ✅ 正确
    pass
# def func(a=10, b):       # ❌ 错误：默认参数后面不能有普通参数
```

⚠️ **坑：默认参数不要用可变类型（如列表）！**

```python
# ❌ 错误写法：每次调用都共享同一个列表对象
def bad_append(item, lst=[]):
    lst.append(item)
    return lst

print(bad_append(1))   # [1]
print(bad_append(2))   # [1, 2] ← 没有新建列表，复用了上次的！

# ✅ 正确写法：用 None 作默认值
def good_append(item, lst=None):
    if lst is None:
        lst = []
    lst.append(item)
    return lst
```

## 5. 参数类型 — *args 可变位置参数

用 `*参数名` 接收**任意个位置参数**，函数内部得到一个**元组**。

```python
def sum_all(*nums):
    """计算所有传入数字的总和"""
    print(type(nums))   # <class 'tuple'>
    total = 0
    for n in nums:
        total += n
    return total

print(sum_all(1, 2, 3))         # 6
print(sum_all(10, 20, 30, 40))  # 100
print(sum_all())                # 0（传 0 个也行）
```

## 6. 参数类型 — **kwargs 可变关键字参数

用 `**参数名` 接收**任意个关键字参数**，函数内部得到一个**字典**。

```python
def show_info(**info):
    """显示个人信息"""
    print(type(info))   # <class 'dict'>
    for key, value in info.items():
        print(f'{key}: {value}')

show_info(name='小明', age=18, city='成都')
# 输出：
# name: 小明
# age: 18
# city: 成都
```

### 四种参数顺序（重点！）

定义函数时，四种参数必须**按以下顺序排列**：

```
def func(普通参数, 默认参数, *args, **kwargs)
```

```python
def func(a, b=10, *args, **kwargs):
    print(a, b, args, kwargs)

func(1, 2, 3, 4, x=5, y=6)
# 输出：1 2 (3, 4) {'x': 5, 'y': 6}
```

| 参数类型 | 写法 | 调用方式 | 接收类型 |
|---------|------|---------|---------|
| 普通位置参数 | `a` | `func(1)` | 普通变量 |
| 默认参数 | `a=10` | `func()` 或 `func(5)` | 普通变量 |
| 可变位置参数 | `*args` | `func(1,2,3)` | **元组** |
| 可变关键字参数 | `**kwargs` | `func(x=1,y=2)` | **字典** |

## 7. 返回值

函数可以用 `return` 返回结果；**可以返回多个值**（实际上返回一个元组）。

```python
# 返回单个值
def square(n):
    return n ** 2

print(square(5))  # 25

# 返回多个值（实际是元组）
def min_max(lst):
    return min(lst), max(lst)

result = min_max([3, 1, 9, 5, 2])
print(result)          # (1, 9)
print(type(result))    # <class 'tuple'>

# 用解包接收多个返回值
lo, hi = min_max([3, 1, 9, 5, 2])
print(lo, hi)  # 1 9
```

⚠️ **注意：没有 `return` 或 `return` 后面没有值，函数返回 `None`**

```python
def no_return():
    print('我没有 return')

val = no_return()
print(val)   # None
```

## 8. 变量作用域 — 局部变量与全局变量

| 变量类型 | 定义位置 | 可访问范围 |
|---------|---------|----------|
| **局部变量** | 函数**内部**定义 | 只在该函数内部有效 |
| **全局变量** | 函数**外部**定义 | 整个文件都可以访问 |

```python
x = 100   # 全局变量

def func():
    y = 200   # 局部变量，只在 func 里有效
    print(x)  # ✅ 函数内可以读全局变量
    print(y)  # ✅ 函数内可以读局部变量

func()
print(x)   # ✅ 100
# print(y) # ❌ 报 NameError：函数外访问不了局部变量
```

### global 关键字 — 在函数内修改全局变量

```python
count = 0

def increment():
    global count   # 声明用全局变量，不是新建局部变量
    count += 1

increment()
increment()
print(count)   # 2
```

⚠️ **不加 `global` 直接在函数内赋值，会新建一个局部变量，不会修改全局变量！**

## 9. 匿名函数 — lambda

用一行代码定义一个**简单的匿名函数**，适合作为参数传入其他函数。

```python
# 语法：lambda 参数: 表达式
# 等同于：def func(参数): return 表达式

# 普通函数写法
def square(x):
    return x ** 2

# lambda 写法（等价）
square = lambda x: x ** 2

print(square(5))  # 25
```

### lambda 常配合 sorted/map/filter 使用

```python
# sorted 按指定规则排序
students = [('小明', 85), ('小红', 92), ('小蓝', 78)]
sorted_students = sorted(students, key=lambda s: s[1], reverse=True)
print(sorted_students)
# 输出：[('小红', 92), ('小明', 85), ('小蓝', 78)]

# map 对每个元素做变换，返回迭代器
nums = [1, 2, 3, 4, 5]
doubled = list(map(lambda x: x * 2, nums))
print(doubled)  # [2, 4, 6, 8, 10]

# filter 过滤元素，返回迭代器
evens = list(filter(lambda x: x % 2 == 0, nums))
print(evens)  # [2, 4]
```

| 函数 | 作用 | 返回 |
|------|------|------|
| `map(func, seq)` | 对每个元素执行 func | 迭代器（用 list() 转换） |
| `filter(func, seq)` | 保留 func 返回 True 的元素 | 迭代器（用 list() 转换） |
| `sorted(seq, key=func)` | 按 key 函数的返回值排序 | 新列表 |

## 10. 递归函数

函数**调用自身**，解决"大问题拆成同类型小问题"的场景。必须有**终止条件**，否则无限递归导致程序崩溃。

```python
# 经典案例：计算阶乘 n! = n × (n-1) × ... × 1
def factorial(n):
    if n == 1:        # 终止条件（必须有！）
        return 1
    return n * factorial(n - 1)   # 调用自身，问题缩小

print(factorial(5))   # 120（5×4×3×2×1）
print(factorial(1))   # 1
```

**递归执行过程（n=3）：**

```
factorial(3)
  → 3 × factorial(2)
       → 2 × factorial(1)
                → 返回 1
       → 返回 2 × 1 = 2
  → 返回 3 × 2 = 6
```

⚠️ **两个必要条件：**
1. **终止条件**：递归必须有停下来的时候
2. **规模缩小**：每次调用，问题要比上次小，最终能触达终止条件

## 11. 高阶函数

**接收函数作为参数**，或**返回函数作为结果**的函数，叫高阶函数。Python 内置了几个常用的高阶函数。

```python
# 1. sorted 内置排序
nums = [3, 1, 4, 1, 5, 9, 2, 6]
print(sorted(nums))                    # [1, 1, 2, 3, 4, 5, 6, 9] 升序
print(sorted(nums, reverse=True))      # [9, 6, 5, 4, 3, 2, 1, 1] 降序
print(sorted(nums, key=lambda x: -x))  # 等同于降序

# 2. map 变换（每个元素 ×3）
result = list(map(lambda x: x * 3, [1, 2, 3]))
print(result)  # [3, 6, 9]

# 3. filter 筛选（保留正数）
data = [-1, 2, -3, 4, -5, 6]
positives = list(filter(lambda x: x > 0, data))
print(positives)  # [2, 4, 6]

# 4. reduce 累积（需要导入）
from functools import reduce
total = reduce(lambda x, y: x + y, [1, 2, 3, 4, 5])
print(total)  # 15（等同于 ((((1+2)+3)+4)+5)）
```

| 函数 | 用途 | 是否内置 |
|------|------|:-------:|
| `sorted()` | 排序 | ✅ 内置 |
| `map()` | 逐元素变换 | ✅ 内置 |
| `filter()` | 条件筛选 | ✅ 内置 |
| `reduce()` | 累积计算 | ❌ 需要 `from functools import reduce` |

## 12. 函数文档字符串

用 `"""..."""` 在函数第一行写说明，可以通过 `help()` 或 `.__doc__` 查看。

```python
def add(a, b):
    """
    计算两个数的和。

    参数:
        a: 第一个数
        b: 第二个数
    返回值:
        a 和 b 的和
    """
    return a + b

# 查看文档
help(add)          # 打印完整文档
print(add.__doc__) # 直接输出文档字符串
```

## 13. 函数速查总表

| 知识点 | 核心要点 | 易错点 |
|--------|---------|-------|
| 定义 | `def 名字(参数):` 冒号不能少，函数体缩进 | 函数体为空要用 `pass` |
| 位置参数 | 按顺序传入 | 顺序错会导致逻辑错误 |
| 关键字参数 | `func(key=val)` 顺序随意 | 关键字参数必须在位置参数后 |
| 默认参数 | `def f(a=10)` 调用时可省 | 默认参数不能用可变类型（列表/字典） |
| `*args` | 接收任意个位置参数，得到**元组** | 函数内用 `for` 遍历 |
| `**kwargs` | 接收任意个关键字参数，得到**字典** | 函数内用 `.items()` 遍历 |
| 返回值 | 可以返回多个值（本质是元组） | 无 `return` 则返回 `None` |
| 作用域 | 局部变量函数外无法访问 | 要修改全局变量需加 `global` |
| lambda | `lambda 参数: 表达式` 一行函数 | 只能写表达式，不能写多行语句 |
| 递归 | 函数调自身，必须有终止条件 | 缺终止条件 → 无限递归 → 崩溃 |

---

## 今日总结

- **函数定义**：`def` 关键字，参数列表可以为空，函数体必须缩进
- **参数顺序**：普通参数 → 默认参数 → `*args` → `**kwargs`，顺序不能乱
- **默认参数陷阱**：不要用可变类型（列表/字典）做默认值，用 `None` 代替
- **返回值**：`return` 可以返回多个值（元组），无 `return` 则返回 `None`
- **作用域**：局部变量只在函数内有效；要修改全局变量必须加 `global`
- **lambda**：一行匿名函数，常配合 `sorted`/`map`/`filter` 使用
- **递归两件事**：终止条件 + 规模缩小，缺一不可
- **高阶函数四件套**：`sorted`（排序）、`map`（变换）、`filter`（筛选）、`reduce`（累积）
