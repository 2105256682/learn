20. 元组是什么？
元组（tuple）是 Python 中另一种容器型数据类型，用圆括号 () 表示。

元组和列表的核心区别：

对比项	列表（list）	元组（tuple）
符号	[]	()
可变性	✅ 可变 — 能增删改	❌ 不可变 — 定义后不能改
速度	相对慢	更快（创建效率高约 8 倍）
线程安全	需要加锁	天然安全（不可变）
用法	数据会变化时	数据固定不变时
⚠️ 元组一旦定义，不能添加、不能删除、不能修改元素，否则报 TypeError。

21. 定义元组和基本运算
定义元组
python
复制
# 三元组
t1 = (35, 12, 98)

# 四元组（可混合类型）
t2 = ('骆昊', 45, True, '四川成都')

print(type(t1))  # <class 'tuple'>
print(type(t2))  # <class 'tuple'>
⚠️ 重要：单元素元组的逗号陷阱

python
复制
a = ()            # 空元组
print(type(a))    # <class 'tuple'>

b = ('hello')     # ❌ 这不是元组！这是字符串
print(type(b))    # <class 'str'>

c = (100)         # ❌ 这不是元组！这是整数
print(type(c))    # <class 'int'>

d = ('hello', )   # ✅ 加了逗号 → 一元组
print(type(d))    # <class 'tuple'>

e = (100, )       # ✅ 加了逗号 → 一元组
print(type(e))    # <class 'tuple'>
记忆口诀：单元素元组必须带逗号，否则括号只是改变优先级。

22. 元组的操作（5 种）
元组支持的运算和列表基本一样，只是不能修改内容：

1. 下标和切片 — 查找、截取
python
复制
t = ('骆昊', 45, True, '四川成都')

# 索引（下标从 0 开始）
print(t[0])     # 骆昊
print(t[-1])    # 四川成都（倒数第1个）

# 切片
print(t[:2])    # ('骆昊', 45)        取前2个
print(t[::3])   # ('骆昊', '四川成都') 每隔2个取1个
2. index() — 返回指定数据的下标
python
复制
t = (35, 12, 35, 87)
print(t.index(35))  # 输出：0（第一次出现的下标）
# 不存在会报错 ValueError
3. count() — 统计出现次数
python
复制
t = (35, 12, 35, 87, 35)
print(t.count(35))  # 输出：3
print(t.count(12))  # 输出：1
4. len() — 获取长度（内置函数）
python
复制
t = (35, 12, 98)
print(len(t))  # 输出：3
5. for 循环遍历
python
复制
t = (35, 12, 98)
for elem in t:
    print(elem)
# 输出：
# 35
# 12
# 98
23. 元组支持的通用运算
这些操作对列表、元组、字符串都适用（因为它们都是序列）：

成员运算（in / not in）
python
复制
t1 = (35, 12, 98)
print(12 in t1)          # True（存在）
print(99 in t1)          # False（不存在）

t2 = ('骆昊', 45, True, '四川成都')
print('Hao' not in t2)   # True（不存在）
拼接运算（+）
python
复制
t1 = (35, 12, 98)
t2 = ('骆昊', 45, True, '四川成都')
t3 = t1 + t2
print(t3)  # (35, 12, 98, '骆昊', 45, True, '四川成都')
比较运算（== / >= / <=）
python
复制
t1 = (35, 12, 98)
t3 = (35, 12, 98, '骆昊', 45, True, '四川成都')

print(t1 == t3)             # False（长度不同不相等）
print(t1 >= t3)             # False
print(t1 <= (35, 11, 99))   # False（逐位比较，12 > 11）
24. 打包和解包
打包（多个值 → 一个变量）
用逗号分隔的值赋给一个变量时，自动打包成元组：

python
复制
a = 1, 10, 100
print(type(a))  # <class 'tuple'>
print(a)        # (1, 10, 100) ← 自动变成元组
解包（一个变量 → 多个值）
把元组赋给多个变量时，自动拆分：

python
复制
a = 1, 10, 100
i, j, k = a      # 解包：三个变量分别接收三个值
print(i, j, k)   # 1 10 100
⚠️ 变量个数必须匹配，否则报错：

python
复制
a = 1, 10, 100, 1000
# i, j, k = a       # ValueError: too many values to unpack（值太多）
# i, j, k, l, m = a  # ValueError: not enough values to unpack（值不够）
星号表达式 * （接收剩余元素）
当变量个数少于元素个数时，用 * 让一个变量接收剩下的所有值（变成列表）：

python
复制
a = 1, 10, 100, 1000

i, j, *k = a
print(i, j, k)        # 1 10 [100, 1000]  ← k 接收剩下2个

i, *j, k = a
print(i, j, k)        # 1 [10, 100] 1000   ← j 接收中间2个

*i, j, k = a
print(i, j, k)        # [1, 10] 100 1000    ← i 接收前2个

i, j, k, *l = a
print(i, j, k, l)     # 1 10 100 [1000]     ← l 接收最后1个（空也可以）

i, j, k, l, *m = a
print(i, j, k, l, m)  # 1 10 100 1000 []    ← m 为空列表
⚠️ 星号表达式的两个规则：

带 * 的变量会变成列表
星号表达式在解包中只能出现一次
解包适用于所有序列（不只是元组）
python
复制
# range 也能解包
a, b, *c = range(1, 10)
print(a, b, c)  # 1 9 [2, 3, 4, 5, 6, 7, 8]

# 列表也能解包
a, b, c = [1, 10, 100]
print(a, b, c)  # 1 10 100

# 字符串也能解包
a, *b, c = 'hello'
print(a, b, c)  # h ['e', 'l', 'l'] o
25. 交换变量的值
Python 中交换变量不需要中间变量，一行搞定：

两变量交换
python
复制
a, b = 10, 20
a, b = b, a      # 一步交换
print(a, b)      # 20 10
三变量轮换
python
复制
a, b, c = 1, 2, 3
a, b, c = b, c, a  # b→a, c→b, a→c
print(a, b, c)    # 2 3 1
原理：Python 字节码有专门的 ROT_TWO 和 ROT_THREE 指令，效率极高。超过三个变量则通过打包/解包完成。

26. 元组和列表的转换
两种类型可以互相转换：

python
复制
# 元组 → 列表
infos = ('骆昊', 45, True, '四川成都')
lst = list(infos)
print(lst)  # ['骆昊', 45, True, '四川成都']

# 列表 → 元组
frts = ['apple', 'banana', 'orange']
tp = tuple(frts)
print(tp)  # ('apple', 'banana', 'orange')
27. 为什么有了列表还需要元组？
原因	说明
线程安全	不可变类型多线程环境下无需加锁，降低同步开销
速度快	创建元组比列表快约 8 倍（见下方测试）
数据保护	有些数据不该被改变（如坐标、常量），用元组防止误改
字典键	元组可以作为字典的键，列表不行（因为列表可变不可哈希）
速度对比实验
python
复制
import timeit

# 创建列表执行 1000 万次
print('%.3f 秒' % timeit.timeit('[1, 2, 3, 4, 5, 6, 7, 8, 9]', number=10000000))
# 约 0.6 秒左右（不同电脑有差异）

# 创建元组执行 1000 万次
print('%.3f 秒' % timeit.timeit('(1, 2, 3, 4, 5, 6, 7, 8, 9)', number=10000000))
# 约 0.08 秒左右（快很多！）
28. 序列通用操作总结
列表、元组、字符串都是有序序列，都支持以下操作：

操作	写法	示例
索引	seq[i]	t[0] → 第1个元素
切片	seq[起始:结束:步长]	t[::2] → 隔一个取
长度	len(seq)	len(t) → 元素个数
成员判断	x in seq	12 in t → True/False
拼接	seq1 + seq2	(1,2) + (3,4) → (1,2,3,4)
比较	==, >=, <=	(1,2) == (1,2) → True
遍历	for x in seq	逐个取出每个元素
今日总结
元组 = 不可变的列表，用 () 定义，定义后不能增删改
单元素元组必须加逗号 (100, )，否则不是元组
元组支持的操作：索引、切片、index()、count()、len()、for 遍历、成员运算、拼接、比较——跟列表一样多，就是不能改
打包与解包：逗号分隔自动打包成元组；赋给多个变量自动拆开；* 星号表达式可以接收剩余元素
变量交换：a, b = b, a 一行搞定，Python 特色语法
为什么需要元组：更快、更安全（线程）、保护数据不被修改
序列通用性：列表、元组、字符串都是序列，共享索引/切片/遍历等操作
